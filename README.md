# Capsule-DQN
DQN using Capsule Network.

## Installation
```
$ git clone https://github.com/Yusuke-Ikezaki/Capsule-DQN.git
```

## How to use

### Simple ver
```
$ python main.py
```

### Custom ver
```
$ python main.py --PARAM1=VALUE1, --PARAM2=VALUE2, ...
```

#### Example
```
$ python main.py --episode=10000, --restore=False
```
